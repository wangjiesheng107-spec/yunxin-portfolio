# Design System

## Visual Direction

Professional, modern, calm, and portfolio-focused. Suitable for HR, product, AI tools, and internship applications.

## Color Palette

- Background: `#f7f4ee`
- Surface: `#ffffff`
- Text: `#1f2933`
- Muted text: `#6b7280`
- Primary: `#215c5c`
- Secondary: `#e7b96f`
- Accent: `#7a5cff`
- Border: `#e5e0d8`

## Typography

Use system fonts. Keep headings confident and compact. Use body text at readable sizes with generous line height.

## Spacing

Use a 4px or 8px spacing scale. Keep sections airy on web and denser in PDF.

## Cards

Use clean cards with subtle borders, 8px or smaller radius, and restrained shadows.

## Buttons

Use clear primary and secondary buttons. Prefer icon + text when available.

## Tags

Use compact tags for skills, methods, tools, and project keywords.

## Data Cards

Use metric cards for project count, internship count, focus area, and measurable outcomes.

## Web Layout Rules

Use a responsive one-page portfolio with strong navigation, project cards, and clear CTAs.

## PDF Layout Rules

Use A4 pages, strong headings, clear page breaks, concise paragraphs, and avoid hover-dependent information.

## Do Not Copy

Do not copy reference logos, brand names, original text, copyrighted illustrations, exact identities, or pixel-perfect layouts.
